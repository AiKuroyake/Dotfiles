## Boot it!

I dual-boot Win10 and Linux. But after installing Pop! OS the computer boots directly to Pop! OS. I fixed it using following commands:

`sudo apt update`

`sudo apt install os-prober`

`sudo os-prober`

`sudo update-grub`
